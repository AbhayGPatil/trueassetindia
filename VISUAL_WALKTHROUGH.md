## 🔄 Data FLOW

### Featured Properties Display
```
1. Page loads → Auth check
2. If not logged in → Show LuxuryPage
3. Component mounts → loadFeaturedProperties()
4. Query Firestore:
   - First attempt: All properties (no filter)
   - If permission error: Retry with status=active
5. Map properties to PropertyCard components
6. Cards render with data from Firebase
7. User can click to view full details
```

### Dynamic Data Shown
```
From Database:
  ✓ Property images (from Firebase Storage)
  ✓ Title, location, description
  ✓ Price, BHK, area
  ✓ Type (Rent or Sale)
  
FormattedPrice Function:
  ₹0-99,999: ₹X format
  ₹100K+: ₹XL (Lakhs)
  ₹10M+: ₹XCr (Crores)
```
